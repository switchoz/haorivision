## 🌟 Brand Essence

### Core Philosophy

> _"Wear the Light. Become the Art."_

HAORI VISION существует на границе искусства, моды и духовности. Мы создаём не одежду — мы создаём **артефакты света**, которые проявляются только в определённых условиях, как внутренний свет человека.

### Visual DNA

```
Японская каллиграфия + Cyber-органика + UV-мистицизм
```

**Ключевые характеристики:**

- **Минимализм** — много пустого пространства (чёрного)
- **Контраст** — резкое противопоставление тьмы и света
- **Таинственность** — не показывать всё сразу
- **Ритуальность** — каждый элемент имеет смысл
- **Технологичность** — современные шрифты, но с душой

### Logo Usage Rules

**DO:**

✅ Maintain clear space

❌ Change colors (except approved variations)
❌ Add effects (except UV glow in digital)
❌ Place over busy images
❌ Rotate or skew

## ✍️ Typography

### Philosophy

> _"Современность с душой. Геометрия с человечностью."_

Мы используем **два шрифтовых семейства**:

1. **Sans-serif** — для структуры, технологичности
2. **Handwritten/Brush** — для души, ритуальности

---

### Primary Typeface: **Neue Haas Grotesk Display**

**Designer:** Christian Schwartz (remake of Helvetica)
**Foundry:** Commercial Type
**Weights used:**

- Light (300) — body text
- Regular (400) — secondary headings
- Bold (700) — emphasis
- Extra Bold (900) — primary headings, logo

**Why Neue Haas Grotesk?**

- Clean, modern, timeless
- Slightly warmer than Helvetica
- Excellent legibility at all sizes
- Premium feel without being pretentious

**Alternatives (if Neue Haas unavailable):**

- Helvetica Neue
- Inter (free Google Font)
- Space Grotesk (free, more geometric)

---

### Secondary Typeface: **Shippori Mincho** (Japanese serif)

**Designer:** FONTDASU
**Foundry:** Google Fonts (free)
**Weights used:**

- Regular (400) — Japanese text, quotes
- SemiBold (600) — emphasis

**Why Shippori Mincho?**

- Authentic Japanese feel
- Works well in Latin and Japanese scripts
- Free and accessible
- Pairs beautifully with geometric sans

**Usage:**

- Japanese characters (光, 闇, 詩)
- Pull quotes from Hikari
- Poetic phrases ("Wear the Light")
- Credits ("Designed by...")

---

### Accent Typeface: **Japanese Brush Script**

**Options:**

- **Shorai Sans** (modern brush calligraphy)
- **Custom hand-lettering** by artist (ideal)

**Usage:**

- Artist signature on haori
- Special edition labels
- Exhibition titles
- Social media quotes

**Rule:** Use sparingly — only when ritualistic, handmade feel is needed.

---

### Type Scale

Based on **1.250 (Major Third) scale**:

```
Hero (H1):       64px / 4rem   | Extra Bold | Letter-spacing: +2%
Heading (H2):    51px / 3.2rem | Bold       | Letter-spacing: +1%
Subheading (H3): 41px / 2.56rem| Bold       | Letter-spacing: 0%
Large (H4):      32px / 2rem   | Regular    | Letter-spacing: 0%
Body (P):        20px / 1.25rem| Light      | Letter-spacing: 0%
Small:           16px / 1rem   | Light      | Letter-spacing: 0%
Tiny:            13px / 0.8rem | Regular    | Letter-spacing: +3%
```

**Line Height:**

- Headings: 1.2
- Body: 1.6
- Captions: 1.4

---

### Typographic Treatments

#### **1. Glow Effect** (UV Mode)

```css
.uv-glow {
  color: #ffffff;
  text-shadow:
    0 0 10px #b026ff,
    0 0 20px #b026ff,
    0 0 30px #ff10f0;
}
```

#### **2. Uppercase + Tracking** (Labels)

```css
.label {
  text-transform: uppercase;
  letter-spacing: 0.15em;
  font-size: 13px;
  font-weight: 400;
}
```

#### **3. Japanese Accent**

```css
.japanese {
  font-family: "Shippori Mincho", serif;
  font-weight: 600;
  color: #b026ff;
}
```

## 🌀 Patterns & Textures

### Philosophy

Паттерны HAORI VISION — это **не декорация**, а **визуальные метафоры**:

- Mycelium network (связь)
- Void bloom (рождение из пустоты)
- Light rays (распространение энергии)

---

### Pattern 1: **Mycelium Network**

**Concept:** Органическая сеть грибного мицелия — невидимые связи под поверхностью.

**Visual:**

```
Тонкие линии (0.5px-1px)
Цвет: #333333 (тёмно-серый на чёрном) или UV Violet
Стиль: Разветвляющиеся, органические кривые
Плотность: Sparse (5-10 линий на 100px)
```

**Usage:**

- Packaging tissue paper (printed in silver)
- Website background (subtle, dark on black)
- Exhibition walls (projected in UV)

**File:** `patterns/mycelium-network.svg`

---

### Pattern 2: **Void Bloom**

**Concept:** Концентрические круги, расходящиеся от центра — как цветение или волны света.

**Visual:**

```
Концентрические окружности
Цвет: Gradient от #000000 (center) → #B026FF (outer)
Стиль: Тонкие линии (1px), 10-15 кругов
Spacing: Увеличивается от центра
```

**Usage:**

- NFT certificate backgrounds
- Product photography backdrops
- Social media story templates

**File:** `patterns/void-bloom.svg`

---

### Pattern 3: **Light Rays**

**Concept:** Радиальные лучи света, как от источника — символ распространения энергии.

**Visual:**

```
Радиальные линии от центральной точки
Цвет: #FFFFFF с opacity 5-10%
Стиль: Тонкие (0.5px), 24-36 лучей
Длина: Варьируется (создаёт ритм)
```

**Usage:**

- Logo backdrop (subtle)
- Presentation slides
- Email headers

**File:** `patterns/light-rays.svg`

---

### Pattern 4: **Grain Texture**

**Concept:** Тонкий noise/grain эффект — добавляет органичность цифровым поверхностям.

**Visual:**

```
Fine grain (1px particles)
Opacity: 3-5%
Цвет: #FFFFFF on black, #000000 on white
```

**Usage:**

- Website backgrounds (всегда)
- Digital images (overlay)
- Video overlays

**File:** `textures/grain.png` (tileable, 256x256px)

---

### Pattern 5: **Kanji Grid**

**Concept:** Тонкая сетка с иероглифом 光 (hikari) повторяющимся — как водяной знак.

**Visual:**

```
Grid: 200px × 200px
Character: 光 (hikari)
Цвет: #0A0A0A (едва заметный на чёрном)
Opacity: 5%
Rotation: 45°
```

**Usage:**

- Packaging interiors
- Certificate backgrounds
- High-end print materials

**File:** `patterns/kanji-grid.svg`

---

### Texture Application Rules

**DO:**
✅ Keep subtle (opacity 3-10%)
✅ Use on large surfaces (backgrounds, packaging)
✅ Combine patterns (e.g., grain + mycelium)

**DON'T:**
❌ Overpower content
❌ Use multiple bold patterns together
❌ Apply to small UI elements (buttons, cards)

## 📷 Photography Style

### Core Principles

1. **High Contrast** — deep blacks, bright highlights (no mid-tones)
2. **Dual Reality** — always show daylight AND UV versions
3. **Cinematic** — 16:9 or 2.39:1 aspect ratio, film-like grain
4. **Intimate** — close-ups, human connection, not catalog-style
5. **Atmospheric** — smoke, haze, light beams

---

### Lighting Guidelines

#### **Daylight Mode:**

```
Lighting: Soft natural light or single key light (45°)
Shadows: Deep, contrasty (film noir style)
Background: Pure black (#000000) or dark texture
Color grading: Desaturated, cool tones (-20% saturation)
```

#### **UV Mode:**

```
Lighting: 365nm UV lamps only (no ambient light)
Background: Absolute darkness
Colors: Vibrant, saturated UV pinks/greens/blues
Glow: Visible halo around fluorescent areas
```

---

### Composition Rules

**Rule of Thirds + Central Focus:**

- Subject slightly off-center
- Light Mark or UV glow at intersection point
- Negative space (black)占 40-60% кадра

**Shot Types:**

1. **Hero Shot** — model in haori, 3/4 angle, looking away
2. **Detail Shot** — close-up of UV pattern (macro lens)
3. **Process Shot** — artist painting, brush in motion (slow shutter)
4. **Transformation Shot** — split-screen daylight/UV comparison
5. **Context Shot** — haori in environment (gallery, club, nature)

---

### Color Grading Presets

**Preset 1: "Void Black"** (Daylight)

```
Shadows: Crushed blacks (-100)
Highlights: Slightly lifted (+20)
Saturation: -30%
Temperature: Cool (-10)
Grain: Fine (5%)
```

**Preset 2: "UV Bloom"** (UV Mode)

```
Shadows: Pure black (-100)
Highlights: Boosted (+40)
Saturation: +50% (only in UV spectrum)
Glow: Bloom effect (10-15%)
Grain: None (digital clean)
```

**Preset 3: "Ritual"** (BTS Process)

```
Shadows: Slightly lifted (-80)
Midtones: Warmed (+5)
Grain: Heavy (15-20%, film-like)
Vignette: Subtle (10%)
```

---

### File Formats & Specs

**For Website:**

- Format: WebP (with JPG fallback)
- Resolution: 2400px wide (retina)
- Compression: 80-85% quality
- Color space: sRGB

**For Print:**

- Format: TIFF or PSD
- Resolution: 300 DPI minimum
- Color space: Adobe RGB (1998)
- Bit depth: 16-bit

**For Social Media:**

- Instagram Feed: 1080x1080px (square) or 1080x1350px (portrait)
- Reels/TikTok: 1080x1920px (9:16)
- Stories: 1080x1920px (9:16)
- Format: MP4 (video), JPG (image)

## 📦 Applications

### Packaging System

#### **Box (Outer)**

```
Material: Rigid cardboard (2mm thick)
Color: Matte black (#000000)
Finish: Soft-touch coating
Logo: Foil stamped (silver #C0C0C0)
Size: 40cm × 30cm × 10cm
Inside: UV pattern printed in UV-reactive ink
```

#### **Tissue Paper**

```
Material: Acid-free Japanese tissue (17gsm)
Color: White (#FFFFFF)
Pattern: Mycelium network (silver print)
Size: 70cm × 50cm (wraps haori twice)
```

#### **Hang Tag**

```
Material: Black card stock (350gsm)
Shape: Circle (8cm diameter)
Front: Light Mark (盧) foil stamped (UV violet)
Back: Edition number, artist signature
String: Black waxed cotton
```

#### **Certificate**

```
Material: Museum-quality paper (cotton, 300gsm)
Size: A4 (21cm × 29.7cm)
Layout: Centered text, wide margins
Elements:
  - Collection name
  - Edition number (e.g., 3/8)
  - Artist signature (hand-signed)
  - Hologram sticker (authenticity)
  - NFT smart contract address
  - QR code → OpenSea listing
Border: Thin silver line (0.5pt)
```
