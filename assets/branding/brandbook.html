Brand Essence

Philosophy

"Свет скрыт в каждом из нас. Хаори даёт ему форму."

HAORI VISION существует на границе искусства, моды и духовности. Мы создаём не одежду — мы создаём артефакты света, которые проявляются только в определённых условиях, как внутренний свет человека.

Visual DNA

Японская каллиграфия + Cyber-органика + UV-мистицизм

Key Characteristics

  • Минимализм — много пустого пространства (чёрного)
  • Контраст — резкое противопоставление тьмы и света
  • Таинственность — не показывать всё сразу
  • Ритуальность — каждый элемент имеет смысл
  • Технологичность — современные шрифты, но с душой

Brand Promise

"Это не одежда. Это артефакт света, который вы носите, коллекционируете и передаёте следующему поколению. Каждое хаори — подписанное произведение искусства с цифровым сертификатом вечности."

Logo System

Логотип состоит из двух элементов: Logotype (HAORI VISION) и Light Mark (абстрактный символ света, напоминающий иероглиф 光).

Primary Logo — Full Version

Use: Website header, packaging, gallery walls

Short Logo — Icon + HV

Use: Social media profile, favicons

Logo Variations

Monogram — Light Mark Only

Use: Labels, embroidery, watermarks

White Background Version

Use: Light backgrounds, print materials

Logo Usage Rules

DO

Use on black or white backgrounds only
Maintain clear space
Scale proportionally
Use official file formats (SVG preferred)

DON'T

Stretch or distort
Change colors (except approved variations)
Add effects (except UV glow in digital)
Place over busy images

Typography Usage Rules

  • ✓ Use sans-serif for structure, technology, products
  • ✓ Use Japanese serif for soul, ritual, quotes from Hikari
  • ✓ Never mix both in the same sentence
  • ✓ Keep line-height 1.6+ for body text (readability)
  • ✓ Use letter-spacing +10-15% for labels and uppercase

Brand Applications

Packaging System

Box (Outer)

  • Material: Rigid cardboard (2mm thick), matte black
  • Finish: Soft-touch coating
  • Logo: Foil stamped (silver)
  • Inside: UV pattern printed in UV-reactive ink

Tissue Paper

  • Material: Acid-free Japanese tissue (17gsm)
  • Color: White
  • Pattern: Mycelium network (silver print)

Hang Tag

  • Material: Black card stock (350gsm)
  • Shape: Circle (8cm diameter)
  • Front: Light Mark foil stamped (UV violet)
  • Back: Edition number, artist signature

Digital Applications

Social Media

Email

Black background, minimal design, Light Rays pattern as header, Hikari tone of voice

Brand Checklist

Before any design leaves the studio, verify:

✓ Design Quality

  • ☐ Uses approved colors only (primary + UV spectrum)
  • ☐ Typography from brand system (Neue Haas Grotesk + Shippori Mincho)
  • ☐ Maintains high contrast (black background + white text)
  • ☐ Logo properly sized with clear space
  • ☐ Patterns used subtly (3-10% opacity)

✓ Brand Spirit

  • ☐ Feels mystical, not commercial
  • ☐ Creates space for reflection (not cluttered)
  • ☐ Minimal design (can we remove anything?)
  • ☐ Hikari would approve (WWHS — "What Would Hikari Say?")
  • ☐ Respects the philosophy: "Свет скрыт внутри"

✓ Technical

  • ☐ Files in correct format (SVG for logos, WebP for web, TIFF for print)
  • ☐ Resolution appropriate (300 DPI for print, 2x for retina screens)
  • ☐ Color space correct (sRGB for digital, Adobe RGB for print)
  • ☐ Accessibility (WCAG AA contrast ratios met)
